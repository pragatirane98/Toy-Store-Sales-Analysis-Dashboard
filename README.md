## 📊 Overview
## Toy-Store-Sales-Analysis-Dashboard
The dashboard provides insights into sales, profit, product performance, and store locations, helping stakeholders make data-driven decisions.

## 🚀 Key Features
📈 Total Sales, Profit & Units Sold KPIs
🧩 Category-wise Sales Analysis
📦 Product-wise Sales Breakdown
🏬 Sales by Store Location (Treemap)
📅 Time Series Analysis with Trend Line
🎛️ Interactive Filters (Year & Store Location)

## 📂 Dataset
The dataset includes:
Sales transactions
Product details
Store location data
Time (date/month/year)

## 📌 Key Insights
🧸 Toys category generates the highest sales (~$5.1M) followed by Art & Craft.
📍 Downtown stores contribute the most revenue whereas as commercial and Residential store location generate revenue $3.28M and $1.66M respectively.
📊 Sales show an upward trend over time
🏆 Top-performing product: Lego Bricks
